Custom Palettes from Images

**Project:** ViewBait.app
**Link:** https://viewbait.app

**The app**
ViewBait generates YouTube thumbnails with saved styles, faces, and color palettes built for contrast at small sizes. Palette Capture lets you define those colors by hand or pull them from a reference image.

**The problem**
Hex pickers work but feel tedious when you already have a logo, brand sheet, or reference thumbnail with the right colors. People guessed swatches and regenerated until contrast felt okay at phone size.

**The solution**
We built **Palette Capture**. Create palettes manually or upload a reference image and let ViewBait extract dominant colors you can edit before saving.

**How it works**
Open My Palettes, upload an image, and the server returns suggested hex values plus an optional name. Tweak the strip, save, and reuse the palette in the generator or chat. Extracted and manual palettes save to the same library and look identical in the dropdown. Starter and above can create custom palettes. Free users browse defaults.

**The result**
Brand colors land in the studio once and repeat across a series without color guesswork every upload. Custom palette creation often follows style extraction and face upload in the same onboarding session.

**Bottom line**
When users already have a visual reference, extraction beats empty forms. Ship both paths and treat AI output as an editable draft. Try it at viewbait.app.
